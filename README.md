
Guía de instalación para Windows:

Descargar git:
https://git-scm.com/download/win

### Clonar el proyecto 
1.Abrir una terminal en la carpeta donde se desea clonar el proyecto
2.Ejecutar el comando:
git clone https://github.com/GeekTaps/Sportify---FICTZ.IT-Demo1


### Instalación del entorno

# instalar .NET 10.0
https://dotnet.microsoft.com/es-es/download

# instalar Node js (versión precompilada para Windows)
https://nodejs.org/en/download

# instalar dependencias de front
1.Abrir una terminal en la carpeta 'frontend' del proyecto
2.Ejecutar los siguientes comandos:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
npm install
